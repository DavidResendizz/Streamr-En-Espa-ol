¿Cómo Funciona?

Una aplicación de ejemplo para Streamr

Streamr proporciona un sistema completo para crear aplicaciones de datos en tiempo real imparables, incluso sin escribir código. Actualmente, la valiosa información que produces está a merced de las grandes corporaciones. En la red Streamr, es suyo para usar y vender como desees, y viaja a través de una red descentralizada peer to peer controlada por nadie e incentivada por el token DATAcoin.

Consideremos un auto eléctrico que se conduce solo. Para un funcionamiento óptimo, necesita datos de otras máquinas, como la información de congestión del tráfico de otros automóviles, los precios de la electricidad en las estaciones de carga cercanas, las previsiones meteorológicas, etc. Streamr proporciona una interfaz única para la entrega y el pago de datos en tiempo real, lo que hace que las transmisiones de datos puedan comercializarse utilizando el token DATAcoin

El automóvil puede obtener de manera autónoma los datos que necesita y pagarlos. A su vez, el automóvil puede vender los datos que produce, como datos de tráfico a otros automóviles, mediciones de la condición de la carretera a una ciudad inteligente, ubicación y nivel de batería para los anunciantes, y así sucesivamente. Nace una economía de flujo de datos.

Motor de Streamr

Los datos brutos a menudo necesitan ser agregados, filtrados o combinados con otros datos para extraer la información y el valor dentro. El Motor está diseñado para los grandes volúmenes de valiosos datos en tiempo real que se originan en dispositivos IoT, intercambios financieros, redes sociales, etc. Los procesos que se ejecutan en el motor pueden, por ejemplo, leer y escribir en blockchains.

Streamr Engine

Raw data often needs to be aggregated, filtered, or combined with other data to extract the information and value within. The Engine is built for the high volumes of valuable real-time data originating from IoT devices, financial exchanges, social media, and so on. Processes running on the Engine can, for example, read from and write to blockchains.

Mercado de Datos

App Store para flujos de datos. Categoriza y muestra flujos de datos disponibles públicamente en Streamr Network, tanto gratuitos como comerciales. El mercado reúne a productores y consumidores de datos, que pueden realizar transacciones utilizando DATAcoin.

Data Market

The App Store for data streams. It categorizes and showcases publicly available data streams on the Streamr Network, both free and commercial. The marketplace brings together data producers and consumers, who can transact using DATAcoin.

Red Streamr

La capa central del sistema Streamr, alimentado por el token DATAcoin. Entrega datos escalables en tiempo real en un patrón de publicación/suscripción a través de una red peer to peer. Streamr Network es una capa de transporte de baja latencia para Dapps en tiempo real y flujos de datos en el mercado. Los nodos de red ganan DATAcoin a cambio del ancho de banda y la validación que proporcionan.

Streamr Network

The core layer of the Streamr System, powered by the DATAcoin token. Scalable real-time data delivery in a publish/subscribe pattern over a peer-to-peer network. Streamr Network is a low-latency transport layer for real-time Dapps as well as data streams on the marketplace. Network nodes earn DATAcoin in exchange for the bandwidth and validation that they provide.

Smart Contracts

La red Streamr existe por encima de la cadena de bloques para velocidad y escalabilidad, pero utiliza Smart Contracts para la mayoría de las operaciones de seguridad crítica, como transferencias de datos DATAcoin, permisos y autogobierno de la red. El Mercado de Datos también utiliza Smart Contracts para implementar modelos de licencias de flujo de datos.

Smart Contracts

The Streamr Network exists above the blockchain for speed and scalability, but uses smart contracts for most security-critical operations such as DATAcoin transfers, permissioning, and network self-governance. The Data Market uses smart contracts as well to implement data stream licensing models.

Ethereum

Ethereum es un blockchain y una plataforma de Smarts Contracts, con la cual los nodos en la red Streamr interactúan regularmente. El motor de Streamr también puede implementar y leer contratos inteligentes y enviar transacciones a la cadena de bloques. Si bien el Sistema Streamr es de un blockchain agnóstico en principio, El primer blockchain compatible será Ethereum.

Ethereum

Ethereum is a blockchain and a smart contract platform, with which nodes in the Streamr Network regularly interact. The Streamr Engine can also deploy and read smart contracts as well as send transactions onto the blockchain. While the Streamr System is blockchain agnostic in principle, the first supported blockchain will be Ethereum.
